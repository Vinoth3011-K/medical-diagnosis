# Medical Diagnosis Chatbot

AI-powered medical diagnosis chatbot using Python Flask backend and React Vite frontend.

## Setup Instructions

### Backend (Python Flask)

1. Install Python dependencies:
```bash
pip install -r requirements_simple.txt
```

2. Run the Flask server:
```bash
python app_simple.py
```

Server will run on http://localhost:5000

### Frontend (React Vite)

1. Install Node.js dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

Frontend will run on http://localhost:3000

## Usage

1. Start both backend and frontend servers
2. Open http://localhost:3000 in your browser
3. Chat with the bot and describe your symptoms
4. Rate your symptoms (0-10 scale)
5. Get AI-powered diagnosis with confidence scores

## Features

- Real-time chat interface
- Symptom rating system
- AI diagnosis with confidence scores
- Probability visualization
- Responsive design

## Note

This is a demo application. Always consult healthcare professionals for medical advice.
